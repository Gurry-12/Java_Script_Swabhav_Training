# 📁 JavaScript Swabhav Training

Welcome to the **JavaScript Swabhav Training** repository. This codebase serves as an organized, structured learning environment mapping my journey through core programming principles, foundational algorithms, and premium interactive web applications built with pure front-end technologies.

The repository is structured sequentially to track educational progress, transitioning from fundamental syntax up to advanced concepts and fluid user interfaces.

---

## 🎯 Learning Objectives

* **Foundational Syntax Mastery**: Explore variable scoping, type conversions, conditional blocks, arrays, and standard loops.
* **Object-Oriented Programming (OOP)**: Implement robust JavaScript classes, inheritances, encapsulation getters/setters, and nested objects.
* **Modern ES6+ Standards**: Utilize spread/rest operators, callbacks, arrow functions, ES6 module imports/exports, and closures.
* **Interactive UI & DOM Manipulation**: Construct responsive glassmorphic interfaces driven by real-time events, key bindings, and native storage state preservation.
* **Defensive Programming**: Learn safe error handling techniques using sandboxed closures and visual try-catch-finally workflows.

## 📂 Repository Structure

The folders in this repository are divided logically by educational categories:

1. **Foundational Principles** (`/01` to `/17`)
   * Core syntax introduction, data types, standard operators, type conversions, conditionals, and looping structures.
2. **Intermediate Logic & Algorithmic Games** (`/18` to `/28`)
   * Number guessing games, arrays, functions, spread/rest operators, 3D dice rollers, arrow functions, and callbacks.
3. **Object-Oriented & ES6+ Concepts** (`/29` to `/39`)
   * Dynamic objects, class inheritances, super keywords, getters/setters, array sorting, shuffles, date objects, and closures.
4. **Time, Dashboards & Advanced Web Systems** (`/40` to `/47`)
   * Asynchronous setTimeout loops, ambient digital clocks, Chronos stopwatches, synchronized financial calculators, modular code separation, sandboxed error handling visualizers, and glassmorphic calculation dashboards.

## 🚀 How to Run the Projects

Since all projects are built as static web assets, running them requires zero installation, bundle compiling, or server setups!

### Method 1: Local Double-Click (Quick Launch)
1. Open the folder of the specific topic you wish to run.
2. Double-click the `index.html` file to launch the page directly in your web browser.

### Method 2: Live Local Server (Hot-Reloading Development)
To host the repository files on a lightweight local server (enabling automatic hot-reloads as you make changes):
```bash
# Install live-server globally
npm install -g live-server

# Spin up the server inside the repository root folder
live-server
```
This launches a development environment accessible at `http://127.0.0.1:8080/` in your browser.
